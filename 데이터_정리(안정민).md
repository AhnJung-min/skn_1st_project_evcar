## 1. 전국 전기차 충전소 표준데이터

- **제공기관**: 한국환경공단
- **포맷**: XML (OpenAPI 제공)
- **End Point**: `https://apis.data.go.kr/B552584/EvCharger`
- **링크**: <https://www.data.go.kr/data/15013115/standard.do>
- **설명**: 지자체·민간이 운영하는 전국 전기차 충전소의 위치·상태·운영정보를 실시간 제공. EV 사용자의 충전 편의 향상 목적.

### 제공 기능 (상세기능 2종, 각 일일 1,000건)

| NO | 상세기능 | 설명 |
|----|----------|------|
| 1 | `/getChargerStatus` | 충전기 상태 실시간 제공 (충전소ID, 충전기ID, 상태정보) |
| 2 | `/getChargerInfo` | 충전소 정보 제공 (충전소명, 위치, 상태, 운영시간, 운영기관명, 충전기 용량) |

### 요청 파라미터

| 항목명 | 설명 |
|--------|------|
| `ServiceKey` | 인증키 (서비스키) |
| `pageNo` | 페이지 번호 |
| `numOfRows` | 한 페이지 결과 수 |
| `period` | 상태갱신 기간 |
| `zcode` | 지역구분 코드 |

### 응답 항목

| 항목명 | 설명 |
|--------|------|
| `resultCode` | 결과코드 |
| `resultMsg` | 결과메시지 |
| `pageNo` | 페이지 번호 |
| `totalCount` | 데이터 총 개수 |
| `busiId` | 기관 아이디 |
| `statId` | 충전소 ID |
| `chgerId` | 충전기 ID |
| `stat` | 충전기 상태 |
| `statUpdDt` | 상태갱신 일시 |

> ℹ️ `getChargerInfo`는 위 항목 외에 충전소명, 주소, 위도/경도, 충전기 타입, 운영시간, 운영기관명, 충전기 용량 등을 추가로 제공합니다.

## 2. 한국도로공사 전국 교통량

- **제공기관**: 한국도로공사
- **포맷**: CSV
- **링크**: <https://www.data.go.kr/data/15062171/fileData.do>
- **보유 파일**: `TCS_31_03_02_218085.csv`
- **설명**: 전국 고속도로 구간의 시간대별 교통량 통계. 차량을 차종별로 세분화하여 집계. 통행 패턴·차종별 도로 이용 행태 분석에 활용.

> ⚠️ **인코딩 주의**: 파일은 **CP949(EUC-KR)** 이며, **헤더가 파일 맨 마지막 줄**에 위치합니다. UTF-8로 읽으면 한글이 깨집니다.
